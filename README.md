# homebrew-moremark

Homebrew tap for [moremark](https://github.com/jasonmimick/moremark) — `more` for markdown, the one that opens a window.

```sh
brew install jasonmimick/moremark/moremark
```

Builds from source on your Mac (needs Xcode Command Line Tools — you have them if you have Homebrew). No unsigned binaries, no Gatekeeper warnings.
